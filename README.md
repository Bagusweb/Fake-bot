# Fake-bot

Ini adalah aplikasi **Fake-bot**, sebuah bot sederhana berbasis web yang memungkinkan pengguna untuk melakukan simulasi percakapan. Aplikasi ini dibangun menggunakan JavaScript dengan antarmuka yang bersih dan mudah digunakan.

## Demo Aplikasi

Kamu bisa mencoba aplikasi **Fake-bot** secara langsung melalui tautan berikut:

[Demo Aplikasi Fake-bot](https://bagusweb.github.io/Fake-bot/)

## Fitur Utama

- Simulasi percakapan dengan respons yang dapat disesuaikan
- Antarmuka yang sederhana dan intuitif
- Desain responsif, dapat digunakan di perangkat seluler maupun desktop
- Respons ditampilkan secara real-time

## Cara Menjalankan

1. Buka tautan [Demo Aplikasi Fake-bot](https://bagusweb.github.io/Fake-bot/).
2. Ketik pesan di kotak chat dan tekan tombol kirim.
3. Bot akan merespons sesuai dengan pesan yang telah ditentukan dalam kode.

## Teknologi yang Digunakan

- HTML
- CSS
- JavaScript
